# Go REST API — Production Grade

A production-ready REST API in Go with PostgreSQL, JWT authentication, bcrypt password hashing, and full CRUD operations.

## Quick Start

### 1. Prerequisites

| Tool       | Version  |
|------------|----------|
| Go         | ≥ 1.22   |
| PostgreSQL | ≥ 14     |

### 2. Clone & Install Dependencies

```bash
# Navigate into project
cd go-rest-api

# Download all dependencies
go mod tidy
```

### 3. Configure Environment

```bash
cp .env.example .env
# Edit .env with your DB credentials and secrets
```

### 4. Create Database & Run Migrations

```bash
psql -U postgres -c "CREATE DATABASE go_rest_api;"
psql -U postgres -d go_rest_api -f migrations/001_init.sql
```

### 5. Run the Server

```bash
go run cmd/server/main.go
```

Server starts at `http://localhost:8080`

## API Endpoints

| Method | Path                  | Auth Required | Description          |
|--------|-----------------------|---------------|----------------------|
| POST   | /api/v1/auth/register | No            | Register new user    |
| POST   | /api/v1/auth/login    | No            | Login, get JWT token |
| GET    | /api/v1/users         | Yes (Admin)   | List all users       |
| GET    | /api/v1/users/:id     | Yes           | Get user by ID       |
| PUT    | /api/v1/users/:id     | Yes           | Update user          |
| DELETE | /api/v1/users/:id     | Yes (Admin)   | Delete user          |
| GET    | /api/v1/profile       | Yes           | Get own profile      |

## Example Requests

### Register
```bash
curl -X POST http://localhost:8080/api/v1/auth/register \
  -H "Content-Type: application/json" \
  -d '{"name":"Alice","email":"alice@example.com","password":"Secret@123"}'
```

### Login
```bash
curl -X POST http://localhost:8080/api/v1/auth/login \
  -H "Content-Type: application/json" \
  -d '{"email":"alice@example.com","password":"Secret@123"}'
```

### Get Profile (with JWT)
```bash
curl http://localhost:8080/api/v1/profile \
  -H "Authorization: Bearer <your_jwt_token>"
```
